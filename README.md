Breaking-Bread Capstone
-This application will allow our customers at Breaking-Bread to customize and add their own orders online! In addition, we also have specialties already set on our application if any customers are interested.

 Breaking-Break Application Features----------------------------------------------------------------------------------------------------------------
- Making new order
- Adding customized sandwiches - You can choose from a variety of our breads, toppings, sides and you can even toast your bread!
- Adding drinks - We have a variety of sodas to choose from!
- Adding chips - There are a variest of chips to choose from!
- Confirming Order - and get your receipt!

In this code, Items is a super class. The getPrice(); method is an abstract method that all of it's child classes are required to have. When this code runs, It gives different outputs depending on what the getPrice(); in that child class is used for. For example, getPrice in sandwich calculates the sandwich using the size and adds up the toppings total. This is different in the drinks class because the getPrice(); in the Drinks class only returns a price based on the size of the drink.
